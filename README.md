This is a simple responsive webpage built using Bootstrap 5.
The project demonstrates how to create a fixed top navigation bar with collapsible menu and search form.

🚀 Features

Responsive Navbar

Fixed Top Navigation Bar

Collapsible Menu (for mobile view)

Bootstrap Search Form

Custom CSS styling (bootstyle.css)

Bootstrap 5 CDN integration

🛠️ Technologies Used

HTML5

CSS3

Bootstrap 5.3

Bootstrap CDN

Preview

The webpage contains:

Brand name on left

Navigation links (Home, Contact, About Us)

Search bar on right

Responsive toggle button for smaller screens

Large content section inside container

⚙️ How to Run

Download or clone the repository.

Open Bootstrap.html in your browser.

Make sure internet is connected (Bootstrap CDN is used).
